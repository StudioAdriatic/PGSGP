# Google Play Games Services Plugin for Godot 
This is an Android Play Games Services plugin for Godot Game Engine

### Supported features:
- Sign-in/Sign out
- Achievements
- Leaderboards
- Events
- Player Stats
- Player Info
- Saved Games
